#  Banking APP with Python

Small Banking Command line application in Python 3 with Mysql Database. 


## App Features

- Create Accounts
- Login
- Deposit Money
- Withdrawal Money
- View Statement
- Generate Statement (CSV)
- Delete Account
- Restore Account 

## Requirnments
- Python 3
- MySql Workbench
- Python IDE (VS Code)


## Functions
- Create Table
- Update Table
- Delete Table
- Insert Table
- Alter Table
- Import and Export CSV files
